Code Challenge BDD
Proje'de BDD framework'ü için Cucumber kütüphanesi kullanılmıştır. Ayrıca proje Maven ve testng ile entegre edilmiştir.
TestRunner.java dosyası üzerinden, Run As TestNG Test ile proje çalıştırılır.
Test raporuna, test çalıştıktan sonra ana dizinde oluşacak olan test-output dosyasındaki emailable-report.html ve index.html üzerinden erişilebilir.
