# Code Challenge BDD
Proje'de BDD framework'ü için Cucumber kütüphanesi kullanılmıştır. Ayrıca proje Maven ve testng ile entegre edilmiştir.
Test Runner.java dosyası üzerinden, Run As TestNG Test ile proje çalıştırılır.
Test raporu için test çalıştıktan sonra test-output dosyasındaki (emailable-report.html) ve (index.html) üzerinden erişilebilir.
